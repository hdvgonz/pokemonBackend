# Ejecutar en Desarrollo

1. Clonar el repo
2. Ejecutar
```
npm install
```
3. Tener nest CLI instalado
```
npm install -g @nest/cli
```

4. Levantar la BD
```
docker-compose up -d
```

## Stack Usado
*MongoDB
*Nest
